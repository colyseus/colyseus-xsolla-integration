Colyseus + Xsolla Payments Integration

This is a simple integration of Colyseus and Xsolla.

  • Client requests a payment token from the Colyseus server
  • The Colyseus server requests a payment token from Xsolla using manually provided user's information

    • (It is recommended to use the @colyseus/auth middleware to authenticate the request instead)

  • Uses the token to open the Xsolla Pay Station UI

This demo does not connect to the database. See database integration recommendations.
